# Configuración del ambiente de desarrollo

## Instalación

Instale las herramientas que usara en el curso. Si usa Windows, **no use espacios en los nombres de las carpetas**

1. Java Development Kit: Descargue el OpenJDK versión 17 <https://adoptium.net/temurin/releases/>

2. Python: Descargue e instale Python versión 3.11 <https://www.python.org/downloads/>

3. Node: Descargue e instale NodeJS versión 18 <https://nodejs.org/en/download/>

4. Maven: Descargue e instale Maven 3.9 <https://maven.apache.org/download.cgi>

5. Gradle: Descargue e instale Gradle 8 <https://gradle.org/install/>

6. Visual Studio Code: Descargue e instale VSC de <https://code.visualstudio.com/>

## Uso

Use este programa para verificar la instalación de las herramientas requeridas en el curso.

1. Clone este repositorio usando el siguiente comando (reemplace URL con la dirección del repositorio)

   ```bash
   git clone URL
   ```

2. Ejecute el comando envcheck

   a. Para Linux y MacOS

      ```bash
      sh envcheck.sh
      ```

   b. Para Windows

      ```bash
      envcheck.bat
      ```
